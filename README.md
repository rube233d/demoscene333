# AI Slop Tsukuyomi / GLVM Opera

Браузерная демосцена про AI slop, сознание и смысл жизни.

[Open the demoscene](https://rube233d.github.io/demoscene333/)

[![Demoscene preview](./README-preview.png)](https://rube233d.github.io/demoscene333/)

## Запуск

```bash
npm install
npm run dev
```

Открыть в браузере `http://localhost:5173`.

## Production preview

```bash
npm run build
npm run preview
```

## GitHub Pages

Для Pages можно публиковать статическую сборку из папки `docs/`.

## Управление

- `Space` - пауза / продолжить
- `R` - рестарт таймлайна
- `M` - mute
- `D` - debug
